## 当前任务

### F-02-1: 前端工程初始化

**类型**: 纯前端  
**优先级**: P0  
**状态**: 待开始

#### 子任务

1. **创建前端工程结构**
   - 使用 Vite + Vue3 + TypeScript 创建工程
   - 目录结构: `frontend/` 或 `src/main/frontend/`
   - 配置 ESLint + Prettier

2. **集成 UI 框架**
   - Element Plus 安装配置
   - 主题色配置
   - 国际化支持（中英文）

3. **配置基础工具**
   - Vue Router 路由配置
   - Pinia 状态管理
   - Axios HTTP客户端配置
   - 环境变量配置（.env.dev/.env.prod）

4. **项目结构搭建**
   ```
   frontend/
   ├── src/
   │   ├── api/           # API接口
   │   ├── components/    # 公共组件
   │   ├── views/         # 页面视图
   │   ├── stores/        # Pinia状态
   │   ├── router/        # 路由配置
   │   ├── utils/         # 工具函数
   │   ├── styles/        # 全局样式
   │   ├── App.vue
   │   └── main.ts
   ├── public/
   ├── index.html
   ├── package.json
   ├── tsconfig.json
   └── vite.config.ts
   ```

5. **添加启动脚本**
   - 开发模式: `npm run dev`
   - 构建: `npm run build`
   - 后端集成构建配置

6. **创建基础页面**
   - Layout布局（侧边栏+头部+内容区）
   - 404页面
   - 首页（Dashboard）

#### 验收标准

- [ ] 工程可正常启动 (`npm run dev`)
- [ ] Element Plus 组件正常使用
- [ ] 路由跳转正常
- [ ] TypeScript 编译无错误
- [ ] 代码格式化配置生效
